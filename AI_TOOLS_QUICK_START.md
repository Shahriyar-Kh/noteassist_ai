## ðŸ› Troubleshooting

### "AI Configuration Required"
- Backend hasn't loaded AI service yet
- **Fix:** Refresh the page
- If persists: Contact admin

### Code won't run
- Syntax error in code
- **Fix:** Check code in editor
- Look at terminal output for error details

### Export PDF fails
- Browser blocked download
- **Fix:** Check browser download settings
- Try uploading to Drive instead

### Google Drive upload fails
- Not authenticated with Google
- **Fix:** Go to Profile > Settings
- Reconnect Google Drive
